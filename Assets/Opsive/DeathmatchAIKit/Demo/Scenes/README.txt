The following assets must be imported for this scene to work:

- Behavior Designer: https://opsive.com/assets/behavior-designer/
- Behavior Designer - Movement Pack: https://opsive.com/assets/behavior-designer-movement-pack/
- Ultimate Character Controller, First Person Controller, or the Third Person Controller: https://opsive.com/solutions/character-solution/
- Behavior Designer / Ultimate Character Controller integration: https://opsive.com/downloads/?pid=803

Before playing the demo scene ensure you have baked the navmesh:

https://docs.unity3d.com/Manual/nav-BuildingNavMesh.html
